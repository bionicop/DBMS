
# Headings:
```

## H2
### H3
#### H4
##### H5
###### H6
```

# Emphasis:
```
Italics - *asterisks* or _underscores_

Bold - **asterisks** or __underscores__.

Strikethrough - ~~OwO~~

Highlights - ==IamUNIQUE!==

Block Quote - > Hello World!

horizontal Rule - 3 Dashes (---)

Coding Blocks - Using Grave Accent (`) 3 times before and after the code[You could even add language name after writing 1st 3 grave accent's without any gap].
```
